# Extend Global (EG) 平台交接文档体系

欢迎来到 Extend Global (EG) B2B2B EOR 平台的系统交接文档库。

本系统是一个支持多层级渠道合作伙伴（Channel Partner, CP）、白标定制、双层发票和四方资金流转的全球名义雇主（EOR）与代理记录（AOR）平台。

为了帮助不同角色的成员在 **3小时内** 快速接手系统，我们构建了以下结构化的文档体系。请根据您的角色，选择相应的入口开始阅读。
