# Extend Global (EG) 邮件品牌色规范

本文档定义了 Extend Global 平台所有邮件通知的品牌色彩体系，适用于 EG 品牌邮件模板（`emailLayout.ts`）和 CP 白标邮件模板（`cpEmailService.ts`）。

## 1. 设计背景

EG 的邮件品牌色从旧版 GEA 的深绿 + 金色配色（`#005430` + `#D4A843`）全面升级为 **Royal Purple** 方案，与 ExtG Logo 的蓝紫渐变风格保持一致。新配色传达出高端、优雅、现代的品牌调性，适合 B2B2B 金融级 HR SaaS 平台的定位。

## 2. 邮件品牌色板 (Email Color Palette)

### 2.1 主品牌色 (Primary Brand Colors)

| 色彩角色 | 色值 | 变量名 | 应用场景 |
|---------|------|--------|---------|
| **品牌主色 (Primary)** | `#6D28D9` | `BRAND_PRIMARY` | 邮件 Header 渐变起点、CTA 按钮、链接色、About 标题色、金额高亮 |
| **品牌浅色 (Primary Light)** | `#8B5CF6` | `BRAND_PRIMARY_LIGHT` | 邮件 Header 渐变终点、Hover 状态 |
| **品牌点缀色 (Accent)** | `#C4B5FD` | `BRAND_ACCENT` | 邮件 Header 下方点缀线（渐变） |
| **品牌深色 (Primary Dark)** | `#3B0764` | — | 备用深色（极端场景） |

### 2.2 Header 渐变

邮件头部使用 CSS 线性渐变，从左上到右下：

```
background: linear-gradient(135deg, #6D28D9 0%, #8B5CF6 100%)
```

> **注意**：由于部分邮件客户端（如 Outlook）不支持 CSS 渐变，代码中使用 `background-color: #6D28D9` 作为 fallback，确保在所有客户端中都能正常显示。

### 2.3 点缀线 (Accent Line)

Header 下方的 3px 点缀线使用渐变：

```
background: linear-gradient(90deg, #C4B5FD, #A78BFA)
```

Fallback 纯色：`#C4B5FD`

### 2.4 文字与背景色 (Text & Background)

| 色彩角色 | 色值 | 变量名 | 应用场景 |
|---------|------|--------|---------|
| 正文主色 | `#1a1a1a` | `TEXT_PRIMARY` | 邮件正文、标题 |
| 正文辅色 | `#555555` | `TEXT_SECONDARY` | Info Card 标签文字 |
| 弱化文字 | `#888888` | `TEXT_MUTED` | Footer、过期提示、时间戳 |
| 页面背景 | `#f4f5f7` | `BG_BODY` | 邮件外层背景 |
| 卡片背景 | `#ffffff` | `BG_CARD` | 邮件内容区域 |
| 分隔线 | `#e5e7eb` | `BORDER_LIGHT` | Info Card 边框、Footer 分隔线 |

### 2.5 语义色 (Semantic Colors)

语义色保持与系统 UI 一致，不受品牌色升级影响：

| 语义 | 色值 | 应用场景 |
|------|------|---------|
| Warning | `#f59e0b` (Amber 500) | 逾期提醒 Banner |
| Danger | `#ef4444` (Red 500) | 失败/逾期状态、紧急 CTA 按钮 |
| Success | `#22c55e` (Green 500) | 成功状态、付款确认 |
| Info | `#3b82f6` (Blue 500) | 信息提示 Banner |

## 5. CP 白标邮件的品牌色处理

CP 白标邮件（`cpEmailService.ts`）**不使用** EG 品牌色，而是动态加载 CP 自身的品牌配置：

| 元素 | 数据来源 | 兜底值 |
|------|---------|--------|
| Logo | `channelPartners.logoUrl` / `logoFileKey` | 显示 CP 公司名文字 |
| Header 背景色 | `channelPartners.brandPrimaryColor` | `#1a73e8`（中性蓝） |
| CTA 按钮色 | 同上 | 同上 |
| 公司名 | `channelPartners.cpBillingEntityName` / `companyName` | — |

## 6. 自定义邮件标签 (Custom Email Tags)

在 `notificationConstants.ts` 的邮件模板中，可使用以下自定义标签，由 `notificationService.ts` 的 `processCustomTags()` 方法解析：

| 标签 | 用途 | 示例 |
|------|------|------|
| `<EG_INFO_CARD>` | 信息卡片容器 | 包裹 `<EG_ROW>` 标签 |
| `<EG_ROW label="..." value="..." />` | 信息行 | `<EG_ROW label="Invoice" value="#001" />` |
| `<EG_BUTTON text="..." href="..." [color="..."] />` | CTA 按钮 | `<EG_BUTTON text="View" href="https://..." />` |
| `<EG_BANNER type="..." text="..." />` | 提示横幅 | `<EG_BANNER type="warning" text="Overdue" />` |
| `<EG_AMOUNT currency="..." amount="..." />` | 大字金额展示 | `<EG_AMOUNT currency="USD" amount="1,500.00" />` |
