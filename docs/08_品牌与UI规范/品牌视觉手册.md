# Extend Global (EG) 品牌视觉与 UI 升级规范

本文档定义了 Extend Global 平台从旧版（GEA 绿/金配色）向全新国际化金融级 SaaS 平台升级的视觉标准。

## 1. 品牌定位与设计理念

Extend Global 是一个面向全球的 B2B2B EOR 金融级 HR 平台。我们的视觉语言必须传达出以下核心特质：

* **专业与可信赖 (Professional & Trustworthy)**：作为处理跨境资金和合规发薪的平台，视觉上不能轻浮。深邃的蓝色系是金融科技的最佳选择。
* **全球化与连接 (Global & Connected)**：通过流畅的线条、地球隐喻或无限延伸的几何图形，展现跨境业务的无缝衔接。
* **现代与高效 (Modern & Efficient)**：抛弃传统的厚重感，采用 Apple Liquid Glass（毛玻璃）设计语言的轻量化变体，让复杂的 B2B 系统显得通透、易用。

## 2. 品牌色彩体系 (Color Palette)

旧版的 GEA 使用了深绿色（Dark Green `#005430`）和金色（Gold `#E1BA2E`），这容易让人联想到传统农业或老派机构。全新的 Extend Global 采用**“深邃靛蓝 (Deep Indigo) + 活力青电 (Electric Cyan)”**的配色方案。

### 2.1 主品牌色 (Primary Colors)
* **EG Indigo (品牌主色)**: `#1E3A8A` (Tailwind `blue-900`)
  * **语义**：权威、合规、资金安全。
  * **应用**：Admin Portal 侧边栏、主按钮、Logo 核心文字。
* **EG Cyan (品牌活力色)**: `#06B6D4` (Tailwind `cyan-500`)
  * **语义**：科技感、速度、全球互联。
  * **应用**：Logo 图标高光、Hover 状态、进度条、图表数据。

### 2.2 界面中性色 (Neutral Colors)
为了配合 Apple Liquid Glass 风格，背景色必须足够干净，以衬托毛玻璃组件的质感。
* **Background**: `#F8FAFC` (极浅的蓝灰，Tailwind `slate-50`)
* **Card Surface**: `#FFFFFF` (纯白，配合 60% 透明度和 20px 模糊)
* **Text Primary**: `#0F172A` (深蓝灰，Tailwind `slate-900`)
* **Text Muted**: `#64748B` (Tailwind `slate-500`)

### 2.3 功能语义色 (Semantic Colors)
* **Success (Active/Paid)**: `#10B981` (Emerald 500)
* **Warning (Pending/Review)**: `#F59E0B` (Amber 500)
* **Destructive (Failed/Overdue)**: `#EF4444` (Red 500)

## 3. 字体排版 (Typography)

保持现有的双字体策略，但调整字重和层级，使其更符合现代 SaaS 的阅读习惯。

* **标题字体 (Headings)**: `DM Sans`
  * 专为数字屏幕设计的几何无衬线字体，字重使用 600 (SemiBold) 或 700 (Bold)，用于页面标题、弹窗标题、大数字展示。
* **正文字体 (Body & UI)**: `Inter`
  * 极佳的阅读性，用于表格数据、表单标签、正文段落。字重使用 400 (Regular) 和 500 (Medium)。
* **等宽字体 (Monospace)**: `JetBrains Mono`
  * 用于显示 API Key、发票编号 (Invoice ID)、精确的金额计算公式。

## 4. UI 升级基线 (UI Upgrade Baseline)

当前系统（基于 `client/src/index.css`）已经引入了 `glass-card` 等 Apple Liquid Glass 设计语言的 CSS 类。本次升级需要将全局的 Tailwind 变量进行替换。

### 4.1 Tailwind CSS 变量替换建议
在未来的代码重构中，`index.css` 的 `:root` 变量应按如下修改：

```css
/* 替换前 (GEA 绿) */
--primary: oklch(0.39 0.094 156.7); /* #005430 */
--accent: oklch(0.80 0.152 92.1);   /* #E1BA2E */

/* 替换后 (EG 蓝) */
--primary: oklch(0.35 0.12 260);    /* Indigo 900 近似 */
--primary-foreground: oklch(0.98 0 0); /* White */
--accent: oklch(0.70 0.15 230);     /* Cyan 500 近似 */
```

### 4.2 白标门户 (App/Worker Portal) 的兜底策略
由于 `App Portal` 和 `Worker Portal` 采用了 `useCpBranding` 动态加载 CP 的品牌色，EG 的主品牌色**不应**直接硬编码在这些门户中。
* **兜底方案**：如果 CP 未配置品牌色，系统默认降级使用 EG 的中性灰蓝色（`#475569` Tailwind slate-600），而不是 EG 的品牌蓝，以保持“白标（White-label）”的绝对中立性。

## 5. Logo 设计理念 (两版方案)

我们为 Extend Global 提供了两个方向的 Logo 设计方案，以满足不同的品牌诉求。

### 方案 A：极简双字母版 (EG)
* **视觉核心**：大写的 `E` 和 `G` 进行几何切割与融合。
* **设计隐喻**：`E` 的中间横线向右延伸，直接穿插进入 `G` 的内部，象征着资金和数据的无缝“跨境流转 (Extend)”。
* **适用场景**：适合作为 App 图标、Favicon、以及界面空间有限的顶栏。

### 方案 B：科技拓展版 (ExtG)
* **视觉核心**：以 `ExtG` 为文字主体，左侧配合一个由经纬线构成的“抽象地球/节点”图标。
* **设计隐喻**：强调 Global 的全球化属性。图标中的节点由中心向外发散，代表 EOR 业务将客户的雇佣能力扩展到全球各地。
* **适用场景**：适合用在官网首页、对外名片、大型发布会 PPT。

*(Logo 的具体视觉呈现见后续生成的设计稿)*
