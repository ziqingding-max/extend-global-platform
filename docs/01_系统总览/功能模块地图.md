# 功能模块地图

本文档梳理了 Extend Global (EG) 平台的核心功能模块及其在三大门户中的分布情况。通过本模块地图，产品经理和开发人员可以快速定位特定功能所属的系统边界与页面入口。

## 1. 核心模块总览

系统的功能架构主要划分为以下六大核心领域：

| 领域 | 核心模块 | 描述说明 |
| :--- | :--- | :--- |
| **基础与组织** | 组织架构、权限管理、CP白标 | 支撑多租户运行的基础设施，包括品牌注入与角色权限控制。 |
| **人力资源 (HR)** | 员工全生命周期、请假管理、报销管理 | 从员工入职、在职管理到离职的完整 HR 业务流。 |
| **财务与结算** | 双层发票、对账引擎、FX剥离、资金流转 | 系统最复杂的部分，处理跨国薪酬结算、多币种转换与利润核算。 |
| **合规与知识库** | 薪酬合规校验、国家指南、AI知识库 | 确保持续符合各国的劳动法与税务要求，提供智能化的合规问答。 |
| **供应商管理** | Vendor网络、账单解析、打款指令 | 管理全球落地的法律实体与落地服务商。 |
| **销售与CRM** | 线索管理、报价单生成、服务合同 | 支撑业务拓展与客户转化流程。 |

## 2. Admin Portal 模块分布

管理后台（`admin.extendglobal.ai`）供 EG 内部运营、财务与系统管理员使用，拥有最高权限。

### 2.1 渠道伙伴管理 (Channel Partners)
* **CP 列表与详情**：管理所有渠道合作伙伴的基础信息。
* **白标配置**：审核并配置 CP 的自定义域名、Logo 与品牌主色调。
* **底层定价配置**：设置 EG 向特定 CP 收取的各项服务底价（Layer 1 Pricing）。
* **CP 钱包管理**：查看 CP 钱包余额，处理线下充值确认与流水记录。

### 2.2 客户与员工管理 (Clients & Workers)
* **全局客户视图**：查看所有 CP 旗下的终端客户列表，支持按 CP 筛选。
* **全局员工视图**：管理所有 EOR 员工与 AOR 承包商的生命周期状态。
* **入职审核**：审核员工提交的入职材料与合规文件。

### 2.3 财务与账单 (Finance Core)
* **发票中心**：管理系统生成的 Layer 1 与 Layer 2 发票，执行手动审核与发布。
* **供应商账单 (Vendor Bills)**：上传并管理当地供应商发来的实际成本账单。
* **双币种对账**：将应收发票（结算币种）与应付账单（当地币种）进行匹配，计算差异。
* **净额法 P&L 报表**：查看剥离汇差与代收代付成本后的真实平台净收入。

### 2.4 系统与合规配置 (System & Compliance)
* **国家指南 (Country Guides)**：维护各国的社保比例、个税规则与雇佣法规。
* **汇率管理**：管理系统使用的基准汇率与对客加价汇率。
* **AI 知识库管理**：上传合规文档，训练并测试内部 AI 问答助手。

## 3. App Portal 模块分布

客户与 CP 门户（`*.extendglobal.ai`）根据登录用户的角色（CP 管理员或终端客户），动态展示不同的功能模块。

### 3.1 渠道伙伴视角 (CP View)
* **我的客户 (My Clients)**：管理 CP 自己签约的终端客户。
* **客户定价 (Client Pricing)**：为不同客户设置特定的服务费加价策略。
* **发票管理 (Invoices)**：
  * 查看并支付 EG 开具的底层成本发票（Layer 1）。
  * 查看并管理向终端客户开具的加价发票（Layer 2），标记客户付款状态。
* **我的钱包 (My Wallet)**：查看当前钱包余额，发起充值申请，或使用余额支付 Layer 1 发票。
* **资金流转看板 (Fund Flow)**：宏观查看收取的客户资金与支付给 EG 的成本之间的差额（Margin）。

### 3.2 终端客户视角 (Client View)
* **人员管理 (People)**：发起新员工或承包商的入职邀请，查看在职人员档案。
* **薪酬与账单 (Payroll & Invoices)**：查看每月的发薪明细，下载并支付由 CP 开具的服务账单。
* **审批中心 (Approvals)**：审批员工提交的请假申请与费用报销单。
* **合规与文档 (Compliance)**：查看员工签署的劳动合同及相关合规文件。
* **成本模拟器 (Cost Simulator)**：预估在特定国家雇佣特定薪资员工的完整企业成本。

## 4. Worker Portal 模块分布

员工门户（`*.extendglobal.ai/worker`）供实际提供劳动的 EOR 员工或 AOR 承包商使用。

### 4.1 个人中心 (Profile & Onboarding)
* **自助入职 (Self-Onboarding)**：填写个人基本信息、紧急联系人、银行收款账户。
* **文档签署 (Documents)**：在线预览并签署劳动合同、保密协议及合规声明。

### 4.2 日常操作 (Daily Operations)
* **工资单 (Payslips)**：查看并下载每月的历史工资单。
* **请假管理 (Leave)**：查看当前可用假期余额，提交年假、病假等休假申请。
* **费用报销 (Reimbursements)**：提交因公产生的费用报销申请，上传相关发票凭证。
* **帮助中心 (Help Center)**：查看常见问题解答，或向平台/HR 提交支持工单。

通过上述模块划分，系统在单一代码库内实现了针对不同用户角色的精准功能投递，确保了业务操作的专注度与数据安全性。
